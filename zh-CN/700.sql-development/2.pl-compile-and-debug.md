# PL 编译和调试

## 背景信息

PL 是一种过程化程序语言（Procedural Language，PL）。是对 SQL 的扩展，在普通 SQL 语句的基础上增加了编程语言的特点，将数据操作和查询语句组织在 PL 代码的过程化代码中，通过逻辑判断、循环等操作实现复杂的功能。

使用 PL 可以编写具有高级功能的程序，能够将业务逻辑封装在数据库内部，提供更好的抽象性和安全性，同时减少了网络的交互，调用速度更快，从而提升整体性能。
